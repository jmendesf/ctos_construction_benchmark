# Benchmark of the construction of the complete tree of shapes

## Presentation
This program aims at comparing two construction pipeline for the complete tree of shapes [1].
The two pipelines are the following:

* Tree of shapes [2] based construction, as introduced in [3];
* Component-tree [4] and tree of shapes based construction, paper currently submitted.

This repository includes the source code for the construction of the complete tree of shapes under both pipelines.

## Requirements
The tree of shapes and component-trees are built using the Higra library [5], which can be downloaded [here](https://higra.readthedocs.io/en/stable/).
